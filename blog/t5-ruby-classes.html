Lets talk about Classes:

So by definition when we are coding we are trying to talk with the computer and explain to it how does our world works. So we need a way to describe as literal as we can the way things are order and classified in our real world to our dumber but quicker silicon friends.

Ruby lets us do that by using the very important Class object. Thing of it like when you where in biology class (the kind of class you had in school), i would guess that your teacher try to explain to you how taxonomy works and how its proposed its to classified the beings that exist in the planet. It does this by starting with the highest level of groups that some beings can be classified and then starts to go down level by level each time disaggregating more each integrant of a group until you get to the most unique being you can have. Quick example:

Ruby classes work in the same way you start with the mother of all the almighty Object class and then start going down form there until you have the most basic objects themselves. Ruby also lets you create your own classes so that you can mimic the real world and explain it to your computer. Example of this:

class Airplanes
def initialize(model, manufacturer)
@model = model
@manufacturer = manufacturer
end
def description
puts "This is a #{model} manufacture by #{manufacturer}"
end
end
A330 = Airplanes.new("A330","Airbus")

In the example above I create the class Airplanes witch groups all airplanes that exist in my universe, it also stores so information about each instance of airplanes( An instance of airplane itâ€™s a fancier ay to say a member or integrant of the class). I also gave my class the power to describe each one of its instances with the instance method (description). So now my computer know that airplanes is a class that stores all airplanes that exist in our program, it also knows the description of each member of airplanes and can give me back that information when I need it, thank you computer you are so kind. And look we also created our first instance of airplanes, the wonderful A330.
